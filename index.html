About The Course

Welcome to SWE 645 - Component-based Software Development! This graduate-level course explores the principles and practices of developing software using component-based architecture.

Course Overview

This course covers essential topics in component-based software development, including:

  • Component Models and Frameworks
  • Service-Oriented Architecture (SOA)
  • Microservices Architecture
  • Cloud Computing and Containerization
  • DevOps and CI/CD Practices

Learning Objectives

By the end of this course, students will be able to:

  • Design and implement component-based systems
  • Deploy applications using cloud services
  • Implement microservices architecture
  • Apply DevOps practices in software development
  • Work with modern development tools and frameworks

Tools & Technologies

  • AWS Cloud Services
  • Docker & Kubernetes
  • Spring Framework
  • Jenkins & GitHub Actions
  • Modern Web Technologies

Course Projects

Throughout the semester, students will work on several hands-on projects:

  • Static Website Deployment on AWS S3
  • Survey Application with AWS EC2
  • Containerized Microservices Development
  • CI/CD Pipeline Implementation
